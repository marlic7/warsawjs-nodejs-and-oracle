Przykład 2a. Pobranie wielu wierszy (oracledb)

var oracledb = require('oracledb');
oracledb.maxRows = 100; // (wartość domyślna)
connection.execute("SELECT * FROM employees", [], { maxRows: 1000 },
                   function(err, result) {
    // result.rows.length <= 1000
});
maxRows - maksymalna liczba wierszy pobieranych przez execute() (gdy nie jest używany ResultSet). Wiersze powyżej tego limitu nie są pobierane.

globalne zwiększenie tego limitu powoduje większą alokację pamięci na poziomie C++ dla każdego execute()
